# 🚀 THE ECONOMIC TIMES – AI Money Mentor

**AI Money Mentor** is a smart financial assistant web application developed to help users manage their finances, plan investments, and make better financial decisions using AI-powered tools.

---

## 🏆 Hackathon Project

This project was developed as a group project for the Economics Times Gen AI Hackathon, showcasing innovative use of AI in personal finance and financial planning.
---

## 🌟 Features

* 🏠 **Home Dashboard** – Overview of financial tools and insights
* 📊 **Portfolio X-Ray** – Analyze your investments
* 🔥 **FIRE Planner** – Plan for Financial Independence & Early Retirement
* 💑 **Couples Planner** – Joint financial planning for couples
* 💸 **Tax Wizard** – Simplify tax calculations and planning
* 🎯 **Life Events Planning** – Plan finances for major life goals
* 🤖 **AI Chat Assistant** – Get smart financial suggestions

## 🚀 Getting Started

### 1. Clone the repository

```id="k8t4we"
git clone https://github.com/your-username/ai-money-mentor.git
```

### 2. Navigate to project folder

```id="d0q8wf"
cd ai-money-mentor
```

### 3. Install dependencies

```id="nsrf7g"
npm install
```

### 4. Run the development server

```id="pb82y8"
npm run dev
```

### 5. Open in browser

```id="4cfbls"
http://localhost:5173/
```
